# Image Review

Use when checking whether a generated or edited asset is acceptable.

## Check
- asset type matches
- art direction matches
- identity is preserved when relevant
- palette, lighting, and composition are correct
- no obvious generation defects
- fits the target resolution and use case

## Steps
1. Compare against the relevant Bible.
2. Compare against canonical references.
3. Approve, edit, or regenerate.
4. Record the outcome in metadata or project memory.
