# Image Review

Use this reference when checking whether a generated or edited asset is acceptable.

## Review criteria
- matches the target asset type
- matches the approved art direction
- preserves character identity when relevant
- uses the correct palette and lighting
- follows the required composition or camera angle
- has no obvious generation defects
- fits the intended game resolution and use case

## Review flow
1. Compare against the relevant Bible.
2. Compare against canonical references.
3. Decide whether the asset is approved, needs editing, or must be regenerated.
4. Record the outcome in metadata or project memory.

## Pass/fail rule
If a critical visual constraint is violated, regenerate or edit again.
