# Prompt Engineering

Use this reference when Claude must translate project knowledge into an image prompt.

## Prompt sources
Build prompts from:
- Style Bible
- Character Bible
- Environment Bible
- UI Bible
- Asset Manifest
- Approved references

## Prompt rules
- Be explicit about asset type.
- Keep stable style constraints at the top.
- Separate required details from optional details.
- Avoid re-describing approved style in conflicting ways.

## Output rule
Prompts should be structured so they can be passed directly to the shell tools.
