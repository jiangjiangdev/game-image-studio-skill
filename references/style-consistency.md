# Style Consistency

Use this reference whenever preserving visual identity matters.

## What must stay stable
- palette
- line weight
- shading style
- perspective
- proportions
- lighting direction
- rendering style
- texture level

## When to load
- Existing character families
- Reused environments
- Asset variants
- Any image that must match approved art direction

## Core rule
The project Bible is the source of truth. If the style already exists, do not re-invent it.

## Validation
Compare the generated asset against canonical references before accepting it.
